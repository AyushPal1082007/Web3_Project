# 🚀 AI Activity Tracker (Blockchain + AI DApp)

A decentralized web application that uses **AI + Blockchain** to track user activities like study, work, exercise, and leisure in a secure and transparent way.

---

## 🌐 Live Demo
👉 https://your-app.vercel.app

---

## 🧠 Project Overview

This project combines:
- 🤖 **AI** → to classify user activities
- 🔗 **Blockchain** → to store data permanently
- 🌍 **Web App** → for user interaction

---

## ⚙️ How It Works

1. User enters an activity (e.g., "I studied today")
2. AI backend classifies it into:
   - Study 📚
   - Work 💼
   - Exercise 🏃
   - Leisure 🎮
3. Smart contract records the activity
4. Counter increases on blockchain
5. UI updates in real-time

## 🔐 Why Blockchain?

- Data is **immutable** (cannot be changed)
- Fully **transparent**
- No need to trust a central authority
- Each action is linked to user wallet

---

## 🤖 AI Logic

Currently uses keyword-based classification:

Example:
- "studied" → Study
- "gym" → Exercise
- "office" → Work

Can be upgraded to real AI models (GPT, Ollama, etc.)
